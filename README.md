# GlitchGame
обучающий пет-проект по созданию игру на unity
## Основная идея 
платфорер головоломка с различными эффетками глюков на разных уровнях
## Идеи для глюков
### 🧠 ГЛЮКИ УПРАВЛЕНИЯ (основа)
1. 🔁 Изменение кнопок
W ↔ S
A ↔ D
WASD → случайные клавиши
кнопки меняются каждые X секунд
2. 🔄 Инверсия
влево = вправо
вверх = вниз
мышка наоборот
3. ⏱ Задержка
input с задержкой (0.2–1 сек)
накапливается очередь действий
4. 👻 Лишние действия
персонаж делает шаг сам
прыгает без команды
5. 🎯 Ограничение
прыжок отключается
### 🎮 ГЛЮКИ ФИЗИКИ
1. 🌍 Гравитация
меняется сила
становится нулевой
переворачивается
2. 🧲 Инерция
персонаж “скользит”
долго останавливается
👁 ВИЗУАЛЬНЫЕ ГЛЮКИ
1. 📺 Экран
тряска
размытие
“шум”
RGB-сдвиг
2. 👻 Объекты
мерцают
исчезают
появляются копии
3. 🎭 Фальшивые элементы
фейковые двери
фейковые враги
### 🧩  ГЛЮКИ УРОВНЯ
1. 🔀 Изменение карты
платформы исчезают
появляются новые
2. 🧱 Коллизии
можно проходить сквозь стены
или наоборот — невидимые стены
3. 🌀 Телепорты
случайные перемещения
зацикленные комнаты
🧠 ГЛЮКИ ЛОГИКИ
1. ⏳ Время
замедление
ускорение
стоп времени
2. 📼 Откат
персонаж возвращается назад
3. 🎲 Рандомизация
разные эффекты каждый раз
### 🧍 ГЛЮКИ ПЕРСОНАЖА
1. 🧬 Дублирование
появляется копия
2. ⚡ Усиление
слишком быстрый
трудно контролировать
